# 알고리즘 스터디 문제 진행 방식 :trollface:
- 해당 주에 풀이할 문제 유형 선택
- 문제 유형에 알맞은 5문제 출제
- 해당 문제 풀이 및 검토
- 해당 문제 솔루션 코드 Public Repository upload :file_folder:
- 코드리뷰 :sweat_drops:
- 솔루션 및 풀이 과정 질문 :raising_hand:
